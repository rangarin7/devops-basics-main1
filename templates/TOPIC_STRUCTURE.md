# The topic tree

- Create a `basic` and `advanced` folder under topic, for example `python` topic, we will have folder structure below:

```
├── python
│   ├── README.md
│   ├── advanced
│   │   └── examples
│   │       ├── 01-factorial-calculator.py
│   │       ├── 02-parse-json-file.py
│   │       ├── 03-oop-with-animal.py
│   │       ├── 04-api-call.py
│   │       ├── README.md
│   │       └── sample_files
│   │           └── persional_info.json
│   └── basic
│       └── helloworld.py
```
